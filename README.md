# micro-fe-demo
Make sure that yarn or npm is installed.
### Project structure: 
- Client: create Counter component then expose at webpack.config.js
- Host: use Counter component of Client remoted at webpack.config.js
### Run: 
- Host
    - cd host
    - yarn / npm install
    - yarn start / npm start
- Client
    - cd remoteclient
    - yarn / npm install
    - yarn start / npm start
